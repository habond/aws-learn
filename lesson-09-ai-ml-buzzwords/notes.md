# Lesson 9 Notes: AI/ML on AWS

## Key Services

### Rekognition (Computer Vision)
- Pre-trained image/video analysis
- Object/scene detection
- Facial analysis and recognition
- Text detection (OCR)
- Content moderation
- Pay per image analyzed

### Bedrock (Foundation Models)
- Access to LLMs (Claude, Llama, etc)
- No infrastructure management
- Pay per token (input + output)
- Fine-tuning capabilities
- Enterprise features

### Comprehend (NLP)
- Sentiment analysis
- Entity recognition
- Key phrase extraction
- Language detection
- Custom entity recognition

### Translate
- Neural machine translation
- 75+ languages
- Custom terminology
- Real-time translation

## Prompt Engineering Tips

### For Bedrock/Claude
- Be specific and clear
- Provide examples
- Use structured output (JSON)
- Iterate and refine
- Use system prompts for context

### Cost Optimization
- Use cheaper models when possible (Haiku vs Sonnet)
- Limit max_tokens
- Cache when appropriate
- Batch requests
